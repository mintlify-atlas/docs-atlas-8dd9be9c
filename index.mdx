# Welcome to CONFOR

CONFOR is a comprehensive forestry accounting system designed for managing forest patrimony, biological assets, and land accounting. Built for forestry professionals and organizations, CONFOR provides powerful tools for tracking and managing forest resources with precision and efficiency.

## Core Capabilities

<AccordionGroup>
  <Accordion title="Hierarchical Forest Structure" icon="sitemap">
    CONFOR implements a five-level hierarchical structure for organizing forest patrimony:
    - **Nivel 2**: Finca, Predio, Hato, Fundo, Hacienda, or ABRAE
    - **Nivel 3**: Compartimiento, Block, Sección, Lote, or Zona
    - **Nivel 4**: Rodal, Parcela, Enumeration, or Unidad de Manejo
    - **Nivel 5**: Referencia, Subunidad, Subparcela, or Muestra
  </Accordion>

  <Accordion title="Role-Based Access Control" icon="shield">
    Comprehensive RBAC system with granular permissions for CREATE, READ, UPDATE, DELETE, EXPORT, and ADMIN operations across all resources.
  </Accordion>

  <Accordion title="Audit Logging" icon="file-lines">
    Complete audit trail with partitioned storage for all system operations including authentication, data modifications, and administrative actions.
  </Accordion>

  <Accordion title="Background Processing" icon="gears">
    Dedicated geospatial worker for processing shapefile imports and recalculating forest areas asynchronously.
  </Accordion>
</AccordionGroup>

## Built With Modern Technology

CONFOR is built with a powerful, modern technology stack:

- **Next.js 16** with App Router for the frontend
- **PostgreSQL** with PostGIS for geospatial data
- **Prisma ORM** for type-safe database operations
- **NextAuth v5** for authentication and authorization
- **Tailwind CSS** for responsive, beautiful interfaces
- **Leaflet** for interactive map visualization
